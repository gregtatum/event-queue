Event Queue Demonstration

This page demonstrates the event queue in JavaScript. The source code is available at event-queue.js

  1. Enable the Firefox Profiler
  2. Start the profile with Ctrl+Shift+1
  3. Start moving your mouse continuously
  4. Click the page to start the test
  5. Keep moving the mouse until the test is done
  6. Capture the profile with Ctrl+Shift+2
  7. View the "Stack Chart" for this page.
  8. View only the JavaScript stacks.
